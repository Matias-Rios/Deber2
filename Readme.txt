NRC :2684
El ejercicio trata de pedir al usuario 2 divisas para
realizar una conversion de estas. Dentro de este debe 
confirmarse que las divisas sean letras y de 3 digitos
y consigo los numeros deben ser mayor 0. Con todo esto 
se realiza la conversion y se imprime un snapshot de mercad
con todos los datos como la divisa, hora y fecha de la consulta
y la conversion de las divisas.
